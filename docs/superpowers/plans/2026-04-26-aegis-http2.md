# Aegis HTTP/2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add real HTTP/2 support to the MITM path and shared upstream HTTPS transport, with explicit multiplexing verification and no policy regressions.

**Architecture:** Keep the existing proxy orchestration in `internal/proxy/server.go`, but replace the manual decrypted-request loop on the MITM path with a handler-driven server path that can speak both `http/1.1` and `h2` over the client-facing TLS connection. Reuse the existing shared upstream transport, but make its HTTP/2 behavior explicit and testable so concurrent requests to one HTTPS origin can share a small number of upstream connections.

**Tech Stack:** Go `net/http`, `golang.org/x/net/http2`, existing Aegis MITM transport path, Go tests, existing perf harness.

---

```go
func TestSharedUpstreamTransportReusesHTTP2Connection(t *testing.T) {
	var (
		mu    sync.Mutex
		conns = map[string]struct{}{}
	)

	upstream := httptest.NewUnstartedServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		mu.Lock()
		if r.TLS != nil {
			conns[r.RemoteAddr] = struct{}{}
		}
		mu.Unlock()
		w.WriteHeader(http.StatusNoContent)
	}))
	upstream.EnableHTTP2 = true
	upstream.StartTLS()
	defer upstream.Close()

	transport := NewUpstreamHTTPTransport()
	transport.TLSClientConfig = &tls.Config{
		MinVersion: tls.VersionTLS12,
		RootCAs:    x509.NewCertPool(),
	}
	transport.TLSClientConfig.RootCAs.AddCert(upstream.Certificate())

	client := &http.Client{Transport: transport}
	var wg sync.WaitGroup
	for i := 0; i < 20; i++ {
		wg.Add(1)
		go func() {
			defer wg.Done()
			resp, err := client.Get(upstream.URL)
			if err != nil {
				t.Errorf("Get() error = %v", err)
				return
			}
			resp.Body.Close()
			if resp.ProtoMajor != 2 {
				t.Errorf("proto major = %d, want 2", resp.ProtoMajor)
			}
		}()
	}
	wg.Wait()

	mu.Lock()
	got := len(conns)
	mu.Unlock()
	if got > 2 {
		t.Fatalf("expected upstream HTTP/2 reuse, got %d remote addrs", got)
	}
}
```

```go
type singleUseListener struct {

func (l *singleUseListener) Accept() (net.Conn, error) {
	var conn net.Conn
	l.once.Do(func() {
		conn = l.conn
		l.conn = nil
```
